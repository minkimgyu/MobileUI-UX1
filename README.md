# Toss Clone
토스 모작 프로젝트

## 프로젝트 소개
토스를 모작하여 개발한 모바일 어플리케이션입니다.



Figma를 사용하여 개발에 필요한 요소들 프로토타이핑 후 React Native를 활용하여 개발 진행을 진행했습니다.
